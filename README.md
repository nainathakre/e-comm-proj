# E-commerce project

### Features 

- Responsive UI
- Log in & Register
- Categories Listing
- Products Listing
- Filtering of products by categories
- Cart

### Local set up

- Clone the repo
- Install dependencies using `npm install`
- Run the project using `npm start`
- Open `http://localhost:3000` for UI, `http://localhost:5000` for api server
